# APK Extractor - Phases 1 & 2

Extract official APKs from Google Play Store with elastic scaling.

## Features

### Phase 1: Single Emulator
- Single Android emulator
- Direct extraction
- Perfect for development/testing

### Phase 2: Docker Containers
- Multiple Android containers
- Load balancing
- Job queue
- Parallel extraction
- Production-ready scaling
